블록체인 기반 탈중앙화 신원증명서비스 과정
1. 개인이 발급기관에 DID 발급요청
2. 발급기관이 블록체인에 발급정보 분산저장
3. 개인이 검증기관에 DID 제츨
4. 검증기관에서 DID 검증
  1. STEP 01 DID 발급

    모바일 인증을 통한 DID 최초발급

    증명서(VC) 발급, 등록

  2. STEP 02 정보 저장

    블록체인에 발급 정보를 분산저장

  3. STEP 03 증명서 제출

    본인 인증을 위해 DID와 함께 필요한 데이터를 선택하여 제출

  4. STEP 04 증명서 검증

    제출된 DID정보를 블록체인 기반으로 검증

  • 데이터 제공사와 무관하게 단일의 ID 사용이 가능하고, 개인이 정보 저장 · 제공 주체가 됨
    * DID(Decentralized Identifiers)는 특정기관이 발급하는 것이 아닌 개인이 중심이 되는 ID
  • VC(Verifiable Credential) 형식에 의해 블록체인이 제공정보의 무결성을 보증
